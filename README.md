# Ourhomes
A society record management system
## Features 
• A database will contain information about the residents of the
society and the registered domestic help.
• It will store the records of the visitors including domestic help
and remaining visitors along with the time of their entry and
exit.
• The guard using this system will be able to :-
       o Record entry/exit of a visitor
       o Seach,add,delete,update the details of
           ▪ Residents
           ▪ Domestic help
       o See the details of every visitor inside at a particular time
       o Review the visitor records for the day
       o Search and view the records of a particular date
       o Search and view the individual entry-exit record of a helper.
