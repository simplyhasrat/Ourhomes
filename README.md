# Ourhomes
A society records management system using mysql and python.
## Features 
+ Uses mysql and python.
+ A database will contain information about the residents of the society and the registered domestic help.
- It will store the records of the visitors including domestic help and remaining visitors along with the time of their entry and exit.
* The guard using this system will be able to :-
     * Record entry/exit of a visitor
     * Seach,add,delete,update the details of
          * Residents
          * Domestic help
     * See the details of every visitor inside at a particular time
     * Review the visitor records for the day
     * Search and view the records of a particular date
     * Search and view the individual entry-exit record of a helper.
